# 🎮 Estudo Pygame - Jogo 2D Arcade

## 💡 Descrição

Este projeto tem como objetivo o **desenvolvimento de um jogo 2D arcade** utilizando **Pygame**, enquanto exploramos funções e algoritmos essenciais para criar a mecânica e interatividade do jogo. O estudo será feito na plataforma **NotebookLM** do Google, onde vamos aprender e aplicar os conceitos de **grafismo**, **transformações geométricas** e **interação com o usuário**.

## 🚀 Objetivos do Estudo

Vamos estudar e implementar:

- 🖼️ **Tela de Abertura** com **algoritmos de rasterização** (reta, circunferência, elipse) e **preenchimento** (Flood Fill ou Boundary Fill).
- 🔲 **Polígonos** com preenchimento por **scanline** e gradientes de cor.
- 🔄 **Transformações geométricas**: translação, escala, rotação e animações.
- 🖱️ **Interação com o usuário** via **teclado** ou **mouse**, além de um **menu interativo**.
- 📏 **Janela e Viewport** com **zoom**, **recorte de Cohen-Sutherland** e transformação de coordenadas.

## 📚 Requisitos Técnicos

- **Polígonos preenchidos com texturas** e gradientes.
- **Transformações geométricas** (translação, escala e rotação).
- **Animações** e **interação com o usuário**.
- **Janela** com transformação de coordenadas de mundo para dispositivo.
- **Viewports** e **zoom** no jogo.
- **Não usar bibliotecas gráficas externas**, apenas as funções básicas do Pygame.

## 🧑‍💻 Tecnologias

- **Pygame**: Biblioteca para criação do jogo 2D.
- **Python 3.13**: Linguagem para lógica e algoritmos.
- **NotebookLM**: Plataforma de aprendizado para explorar o Pygame.
